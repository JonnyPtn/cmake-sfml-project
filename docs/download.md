# Downloading

To get this project onto your machine you can do what's called a "clone" via git

The simplest way to do that on any platform is to open a terminal and run 

`git clone https://github.com/JonnyPtn/cmake-sfml-project`

which will clone this repository contents into a `cmake-sfml-project` subfolder wherever you ran the command

If you are using visual studio, it's possible to clone and set up this project from within the IDE
