# Git

This guide covers how to set up git for SFML development on your computer. There are multiple options for each platform so you can decide what works for you

## Windows

- Use the official windows installer, which you can find [here](https://github.com/git-for-windows/git/releases/download/v2.47.1.windows.1/Git-2.47.1-64-bit.exe)
- From winget, open a windows terminal and run `winget install git.git`
