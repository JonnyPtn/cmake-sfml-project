# CMake

This guide explains how to set up CMake for SFML development on your computer. There are multiple options for each platform so you can decide what works for you

## Windows

- Use the official windows installer, which you can find [here](https://github.com/Kitware/CMake/releases/download/v3.31.2/cmake-3.31.2-windows-x86_64.msi)
- From winget, open a windows terminal and run `winget install cmake`
